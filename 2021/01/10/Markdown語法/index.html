    • Markdown語法

    Hexo是一款使用Markdown語法撰寫的部落格工具
    此篇紀錄常用語法並重啟記憶

    無序清單

    -、+、*等符號後方加上一個空白皆為清單，若要表示下一層級，多一個縮排或兩個空白即可

    有序清單

    數字+.開頭，若要表示下一層級，多一個縮排或兩個空白即可

    強調

    看這邊**我是範例** or 看這邊__我是範例__

    看這邊我是範例

    斜體

    看這邊*我是範例* or 看這邊_我是範例_

    看這邊我是範例

    區塊程式碼

    ```

    var i = 0

    ```

    行內程式碼

    我是範例 ` var i = 0`

    這是一段內容 var i = 0

    連結

    - 前者為 [ ]：中括號內需要補上連結的顯示文字。

    • 後者為 ( )：小括號內補上的是連結路徑。

    輸出後: Google

    圖片

    屬於行內元素，可放在文字段落中，但大多情況下會將圖片作為獨立區塊使用

    • ![ ]：與連結結構接近，但前方緊貼著 ! 符號。中括號的內容也並非必填，其文字內容通常作為 hover 後的提示文字或作為 SEO 增強使用。

    • ( )：圖片連結位置。

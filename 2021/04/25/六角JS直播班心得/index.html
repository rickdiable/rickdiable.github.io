    • 六角JS直播班 心得

    前言

    接續著上一堂網頁切版直播班繼續向著前端深處鑽研，這次參加的是六角的JS直播班。

    課程剛開始時老師分享切版直播班的作業量大約是此次的十倍，才知道原來當時切版班是真的硬(苦笑)，於是訂下這期至少進度不要落後的目標。

    學習過程

    六角學院設計了許多功能及系統幫助學員能夠順利學習，在卡關時也提供多種管道尋求解答，這些設計最終緊密相連在一起，達到了 1 + 1 > 2 的效果。

    • 觀看錄影檔功能

      • 在課程正式開始前，老師準備了許多章節的預習課程供我們了解有關JS的一些基本知識

      • 每週逐步增加知識點以達到能夠完成該週主線任務的能力

    • 每日任務

      • 透過每日任務能夠實際操作補充的知識點

      • 將該週主線任務拆分成小部分來練習，降低複雜程度、增加熟悉度

      • 很適合作為一天學習的開頭，醒醒腦 XD

    • 主線任務

      • 觀念確認、動手實作、增加信心及熟悉度

      • 分級挑戰 : 增加認識程式的廣度及彈性

    • 合作系統

      • JS班群組 : 只要你敢問，甚麼疑難雜症大家都熱心回答

      • 小組 : 小組討論，集思廣益、增加知識點分享

      • 老師及助教 : 連老師都常潛伏其中，最可靠的解惑大隊!

    • 助教分享

      • 助教直播教學主線的挑戰點，或分享常用套件

      • 要做出一個功能，方法能有千百種，看助教分享，增加寫 code 廣度

    • 熱血加碼

      • 洧杰老師全天 Live Demo : 觀摩開發過程、 Debug 方法、學習開發思路

      • 助教任你問 : 問爆!助教們辛苦了!!

    收穫

    • 流程圖

      • 比起刻出主線作業，老師在課程中提到的流程圖、拆解任務及小步測試等，反而是讓我感到獲益良多的部分，這些方法能夠用在許多場合，讓事情處理起來更有條理，增加效率並減少失誤。

    • BMI KATA

      • 很有趣的是課程中接觸了 BMI KATA 這項特別的挑戰，透過不停撰寫同一段程式碼，達到熟悉公式、函式、結構的目的，過程中也體會能透過重構或更簡潔的方式撰寫程式碼。

    BMI KATA 盛況!

    小結

    時光飛快地流逝，轉眼間JS班也到了尾聲，雖然訂下跟上進度的目標，但最後還是有點落了隊。

    課程中參考了非常多同學的作業、筆記，發覺還有太多不會用的 JS 方法，但俗話說
